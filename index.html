PassApp

The new way to have unlimited different passwords and remember only one.

Why this app?

Nowadays one needs to remember dozens of passwords, each of it has to be unique and meet some strength requirements. But that's not always possible.

So there are some ways to master this challenge. One only has a few passwords and uses them multiple times, maybe with some small changes. But if one password is leaked, others are also.

The other possibility is to use a password manager, but it is much work to set it up and if it gets hacked, all passwords are leaked.

How does this app work?

This app generates unlimited unique and strong passwords out of one password. So one only has to remember the main password.

This app uses a one direction algorithm to generate the unique passwords. Each time the main password is added to the use case (e.g. Facebook, Mail) and given to this algorithmus. Just having the output it is impossible to calculate the input data.

All the generated passwords are different and strong, also it is not really possible to guess the password for Facebook if you know the one for Twitter.

The generated passwords always have the same quality, also if your main password is just "test".
But a weak main password can be brute forced if one knows one generated password. (If you know the Twitter password, you can test all main passwords until the generated matches the known.)

How to use the app?

Just take one good password, it should be unique and strong. And you will have to remember that main password.

To get the password for a service you just have to type your main password in the "Main password" field. Also, you have to specify the use case for which the password should be generated (type your own or choose from dropdown). Afterwards the password for the chosen destination can be copied.

Every time you need the password for the use case, you just need your main password and the web app will generate your password again.
